# mcp-servers

A Helm chart to deploy multiple MCP servers using the bjw-s helm library. This chart includes:
- grafana-mcp (official)

## Usage

Customize `values.yaml` to enable/disable subcharts and configure their values.

## Requirements
- Kubernetes >= 1.22
- Helm >= 3.7.0
